## Dependencias :warning:
### iniciando projeto
- npm init
### instalando o express
- npm install express --save
### instalando ejs
- npm install ejs --save
### instalando nodemon
- npm install -g nodemon
### instalando mysql
- npm install mysql --save
### instalando body-parser
- npm install body-parser --save
